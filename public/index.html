📥 Import legacy data

In the OLD single-file app, click ⬇ Backup to download its JSON file, then select it here. Employees, company profile and advances import directly into this database (bank numbers get encrypted). KYC files aren't in the backup — re-upload them per employee.

🏦 Import bank details (from salary sheet)

Upload a CSV with columns Location, Name, Account Holder, Account No, IFSC (one row per employee). Each row is matched to an employee — you review and fix the matches before anything is saved. Applying overwrites the matched employee's bank details (Bank Name is filled from the IFSC).
